This respository contains .qmd template files, data files, and image files for students to use when completing assignments in Villanova's STAT 4380 course.
Course website: https://nova-stat-4380.netlify.app/schedule/
Example: students should use a copy of lab_01.qmd in the labs folder to type their answers to Lab 01, which has instructions available at https://kgfitzgerald.github.io/NOVA_STAT_4380/labs/lab-01-hello-r.html
